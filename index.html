Program

Click here to go to the folder: link

You may either use the batch install.bat or extract the compressed file workbench.7z to C drive. By default the batch will extract all the files to C:\MineStock and logging paths are set to C:\Temp, a shortcut will also create in your desktop which points to C:\MineStock\Program\FYP_GUI_v1\bin\Release\FYP_GUI_v1.exe.

Source codes are also included in this installation package.

Installation package partly created by 7-Zip SFX Maker v3.2: http://teejee2008.wordpress.com
